# psychologist_AI

Школьный психолог на основе искусственного интеллекта

---

## Быстрый старт (первое развёртывание)

**Требования:** Docker, Docker Compose. При первом клонировании: `composer install`.

```bash
chmod +x setup.sh
./setup.sh
```

Скрипт поднимает контейнеры, собирает образы и запускает миграции. После выполнения:
- приложение: http://localhost:9090  
- Adminer (БД): http://localhost:9091  

---

## Что такое Adminer и как в него войти

**Adminer** — это веб-программа для просмотра и правки базы данных приложения. База данных хранит, например, логины, диалоги, настройки. Adminer нужен, когда надо что-то посмотреть или поправить в этих данных вручную.

**Как войти:**
1. Откройте в браузере: http://localhost:9091  
2. На странице входа укажите (скопируйте как есть):

   | Поле       | Значение           |
   |------------|--------------------|
   | Система    | PostgreSQL         |
   | Сервер    | postgres           |
   | Пользователь | psychologist_user |
   | Пароль    | psychologist_pass  |
   | База данных | psychologist_db  |

3. Нажмите «Войти».

После входа вы увидите список таблиц. Без необходимости их лучше не менять.

## Запуск проекта

| Действие | Команда |
|----------|---------|
| Запустить | `docker compose up -d` |
| Остановить | `docker compose stop` |
| Остановить и удалить контейнеры | `docker compose down` |
| Пересобрать и запустить | `docker compose up -d --build` |
| Миграции | `docker compose exec php php bin/console doctrine:migrations:migrate --no-interaction` |
